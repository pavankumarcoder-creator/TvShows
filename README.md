# My Favorite Games Website 🎮
A simple HTML website that showcases my favorite games with images and descriptions.

---

## 📌 About the Project
This project contains a simple multi-page website built using only HTML.  
It lists three favorite games:
- Basketball
- Chess
- Cricket

Each game has its own dedicated page with an image and description.

---

---

## 🖥️ Pages

### 🏀 Basketball
Explains the rules and overview of basketball.

### ♟️ Chess
Explains chess gameplay and rules.

### 🏏 Cricket
Explains cricket rules and formats.

---

## 🚀 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Click on any game to view details
